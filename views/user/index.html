LagerNode Core

Backend API for product management

About the Project

LagerNode Core is a backend API built with Node.js that allows you to manage products. It provides endpoints for working with product data such as name, category, stock and price.

Features

  • Get all products
  • Get product by ID
  • Filter products by category
  • Search products by name
  • Filter products by price range
  • Get low stock products

Purpose

This project was created as a learning backend application to practice API development, routing, data handling and building a clean project architecture.

Tech Stack

  • Node.js
  • Express.js
  • JSON-based data storage
